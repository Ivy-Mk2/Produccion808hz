# Resumen de estructuras y módulos (Productora 808xHz)

| Estructura | Módulo del proyecto | Justificación |
|---|---|---|
| Lista simple | Cotización | Manejo secuencial de items, cálculo de total, ordenaciones y búsquedas |
| Lista doble | Reservas | Navegación día/hora hacia adelante y atrás, inserción/eliminación eficiente |
| Lista circular | Inventario | Rotación continua de equipos disponibles y mantenimiento |
| Pila | Historial de acciones | Undo/Redo sobre cotizaciones y utilidades infija→postfija |
| Cola | Logística | Asignación FIFO de técnicos y equipos, pedidos en espera |
| Árbol binario de búsqueda (ABB) | Clientes / Inventario | Búsquedas rápidas por DNI/ID/nombre y ordenamiento natural |
| Grafo | Rutas / Paquetes | Modela trayectos y dependencias, BFS/DFS/Dijkstra/Floyd‑Warshall |

Este resumen cubre el sílabo y el informe empresarial, vinculando cada estructura a un proceso real del sistema.
